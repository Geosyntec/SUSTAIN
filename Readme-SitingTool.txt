SUSTAIN_SitingTool_Version_1.0.zip
==================================
The zip file contains the source code for the BMP Siting tool of SUSTAIN 1.0. 
The source code is in Visual Basic 6 programming language and requires 
Visual Studio 6 and ArcGIS 9.3.1 software to compile it on Windows 32-bit 
operating system. The project file "BMP_Siting_Tool.vbp" launches the project.
